# 理想气体状态方程实验操作步骤
## 1.准备阶段
在UI界面中选择以下三种实验过程之一：

- **等温过程**：验证玻意耳定律（PV = 常数）
- **等压过程**：验证盖-吕萨克定律（V/T = 常数）
- **等容过程**：验证查理定律（P/T = 常数）
## 2.开始实验
1. 点击"开始实验"按钮
2. 此时会显示参数面板和输入面板

## 3.数据收集
通过以下两种方式之一改变气体状态：
### 方式一：移动活塞
1. 点击并拖动气缸中的活塞
2. 向上拖动活塞：增大体积，减小压强
3. 向下拖动活塞：减小体积，增大压强
4. 系统会自动收集数据点

### 方式二：调整滑动条
1. 根据所选实验过程，调整相应的滑动条：
- 等温过程：调整体积滑动条
- 等压过程：调整温度/体积滑动条
- 等容过程：调整温度滑动条
2. 系统会自动收集数据点

## 数据分析
1. 当收集到足够的数据点后，点击"确认"按钮
2. 系统会自动绘制四种图像：
- P-V 图像（压强-体积）
- P-1/V 图像（压强-体积倒数）
- P-T 图像（压强-温度）
- P-1/T 图像（压强-温度倒数）
3. 系统会分析数据并验证相应的气体定律
4. 可点击"重置"按钮重置实验过程，回到准备阶段

## 注意事项
1. **操作顺序**：请按照实验步骤顺序操作，确保实验的正确性
2. **边界限制**：当活塞或滑动条达到边界时，无法继续向超出边界的方向移动
3. **数据收集**：请确保收集足够的数据点，以获得准确的分析结果
4. **误差分析**：实验结果包含一定的误差，这是正常现象，符合实际实验情况
